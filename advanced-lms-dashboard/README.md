Advanced Learning Management System

Project Overview

Advanced Learning Management System (LMS) built using HTML, CSS, JavaScript, Firebase Authentication, and Firebase Firestore.

Features

- Admin Login
- Student Login
- Admin Dashboard
- Student Dashboard
- Course Management (Add, Edit, Delete Courses)
- Student Management (Add, Edit, Delete Students)
- Reports Dashboard
- Firebase Authentication
- Firebase Firestore Database Integration

Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore

Project Structure

- index.html
- admin-login.html
- admin-dashboard.html
- manage-courses.html
- manage-students.html
- reports.html
- student-login.html
- student-dashboard.html

Future Enhancements
